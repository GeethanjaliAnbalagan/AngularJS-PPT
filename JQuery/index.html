JQuery

This is a test page created to test the useability of JQuery in combination with AngularJS.
By default JQlite is included. This can be overwritten to use full-blown JQuery library's as in the example page.

For styling reasons the Twitter Bootstrap CSS library has been included.

DND (drag-and-drop)

Box A als C can be dragged across the entire page. Box B is only able to recieve draggable boxes and C can be dragged and dropped.
When you hover a draggable box over a droppable one the color of the box will change.

[A] draggable

[B] droppable

[C] Beide

Date picker

uses the JQuery Datepicker in combination with the AngularJS two-way databinding model.
